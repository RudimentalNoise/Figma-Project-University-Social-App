# Figma-Project-University-Social-App
This project was developed as part of a Human-Computer Interaction (HCI) course.
The UniConnect project was created with the goal of developing a single digital platform that allows university students to easily and intuitively manage three fundamental aspects of academic life: sports training, university courses, and canteen services.
Through a consistent and user-friendly interface, the app enables students to connect, organize their activities, and quickly access useful information.
