About the Project

This project was developed as part of a Human-Computer Interaction (HCI) course. It showcases a comprehensive design for a university social application that helps students connect, collaborate, and engage with campus life.

The design focuses on user-centered principles, intuitive navigation, and accessibility to create an inclusive platform for all students.

Key Features

Student Profiles

Create and customize your personal profile to connect with peers

Event Discovery

Find and join campus events, clubs, and activities

Study Groups

Form or join study groups for your courses

Campus News

Stay updated with the latest campus news and announcements
